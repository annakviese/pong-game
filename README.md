# Pong Game

This is a basic pong game that was created using HTML5 Canvas.
The game allows 2 players and has a score board. 
Audio sounds are triggered on ball bounces, goals, and paddles. 

#Setup

####Install

> npm i

####Run

> npm start

# Keys 
##### Player 1:
* a: up
* z: down
* q: fire a new ball

##### Player 2:
* ▲ : up
* ▼: down
* /: fire a new ball
